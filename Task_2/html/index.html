Introduction

This code was developed as a part of as assignment for Logic in Computer Science. It is a simple proof-checker which takes in a proof and outputs if the proof is valid or not.

Details

This code takes in the number of lines in your proof and the lines as inputs, and returns if the proof is valid or not.

Checked

Rules allowed in the proof are : 1) Premise 2) And Introduction 3) And Elimination 4) Or Introduction 5) Implies Elimination 6) Modus Tollens

Compilation

You can compile this code by g++ task_2.cpp and execute it by ./a.out
