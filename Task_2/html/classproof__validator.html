int proof_validator::return_rule_operator_index ( string  line)

Function which returns the root operator of a given expression; i.e. root of parse tree of expression.

Parameters
stringline string which is the line whose root operator we need

The function checks different possible scenarios for the expression which can be received. If the expression still has parantheses after removing the outermost parantheses, it counts upto that index in the string where the number of opening and closing brackets become equal and returns that index. Else it returns 1 or 2 depending on whether the first atom has a negation or not.
